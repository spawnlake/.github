# Support

Need help with SpawnLake or one of the `boid-*` images?

## Documentation

- Product docs: <https://docs.spawnlake.com>
- Per-repo `README.md` and `QUICKSTART.md`

## Questions and discussion

- **GitHub Discussions** on the relevant repository — best for "how do I…?",
  design questions, and showing off what you built.
- **Issues** — only for confirmed bugs and concrete feature requests. Use
  the templates so we have what we need to act.

## Commercial / hosted support

For SLA-backed support on the hosted SpawnLake service, see
<https://spawnlake.com> or email **hello@spawnlake.com**.

## Security issues

Do **not** open a public issue or discussion. See
[SECURITY.md](./SECURITY.md).
