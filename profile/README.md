# SpawnLake

**Sovereign, on-demand container workspaces.** A catalog of curated images
(`boid-*`) plus a control plane (`spawnlake-api`), data plane (`spawnlake-rte`),
gateway, console, CLI, and MCP server — so humans and AI agents can compose,
spawn, and drive isolated workspaces.

- Website: <https://spawnlake.com>
- Docs: <https://docs.spawnlake.com>
- Status: <https://status.spawnlake.com>
